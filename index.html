Uber Fare Prediction

Project Overview

This project uses historical Uber ride data to build a machine learning model that predicts the fare of a trip based on variables such as trip distance, pickup and dropoff locations, time of day, and day of the week.

The dataset used for training the model was sourced from Kaggle: Uber Fares Dataset.
